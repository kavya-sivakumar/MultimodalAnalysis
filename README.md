# MultimodalAnalysis

HRV Synchrony Analysis:
> windowed cross-correlation of cardiac interbeat intervals between dyads </br>
> rolling-window CCF to detect temporal couplling

Movement Synchrony: 
> optical flow based extraction of postural sway from video </br>
> dynamic time warping for phase locked coordination detection

Behaviourial Multimodal Integration:
> synchronised timestamp alignment across EDA, video-coded behaviour, and audio features </br>
> event-triggered segmentataion for multistram pattern detection

Note: </br>
The following project is an exploratory proof-of-concept pipeline built using simulated data.
